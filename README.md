# Simple Interest Calculator

This repository contains a simple Bash script that calculates simple interest from user input.

## Project Description

The simple interest calculator asks the user to enter three values:

- Principal amount
- Annual rate of interest
- Time period in years

After collecting the input values, the script calculates and displays the simple interest.

## Formula

```text
Simple Interest = (Principal x Rate x Time) / 100
```

## Example

```text
Principal = 1000
Rate = 5
Time = 2

Simple Interest = (1000 x 5 x 2) / 100
Simple Interest = 100
```
